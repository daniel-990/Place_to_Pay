#mediante la pasarela de pagos de *[place to play](https://www.placetopay.com/)

 	● Número de referencia
	● Valor a pagar total
	● Descripción del pago a realizar

	● Email
	● Tipo de documento
	● Documento de identidad
	● Nombres
	● Apellidos
	● Teléfono
	● medios de pago
